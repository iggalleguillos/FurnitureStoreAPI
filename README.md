# FurnitureStoreAPI

Simple Rest API made with NodeJS, Koa and MongoDB

### Requeriments 
```
Node v14.5.0
```
### Installation
```
npm install
```
### Run Development Mode
```
npm run dev
```
### Run Test
```
npm run test
```
